🏥 RushMedz App Icons

Click "Download All Icons" to save PNG icons for each app

📋 How to Use These Icons

  1. Download icons for each app by clicking the download button
  2. Extract the ZIP file (or save individual files)
  3. Copy icons to assets folder:
    • icon.png → Main app icon (1024x1024)
    • adaptive-icon.png → Android adaptive icon foreground (1024x1024)
    • splash.png → Splash screen image (2048x2048)
    • favicon.png → Web favicon (48x48)
  4. Rebuild your app with npx expo prebuild --clean then build
